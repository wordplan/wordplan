Integration of WordPress and Megaplan (CRM)

== Description ==
Implementation of a mechanism for flexible exchange of various data between Megaplan and a site running WordPress.

== Translations ==
* English - default, always included
* Russian - always included

== Installation ==
Archive extract and upload folder "wordplan" to /wp-content/plugins (final path: /wp-content/plugins/wordplan).
